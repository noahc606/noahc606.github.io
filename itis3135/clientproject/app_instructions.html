Technical Portfolio Studio - Manual

Welcome

You have reached the instructions page for Technical Portfolio Studio, the revolutionary app that can help you get started building your own personal website.

And... it's completely free.

What is This?

For my client project, I decided to create a website builder for a technical portfolio website. I was inspired by the BYO Intro assignment and several ideas a friend of mine had. Actually, he had envisioned a Java graphical application that would build the HTML code as a string. I suggested to him the idea of making it a web application instead - the benefit of a web app is that the user could have a live view of what their website would look like as it was being built.

How Does It Work?

You can navigate to what will become your very own website(s), by clicking the "My Site" button on the bottom left corner. Once there, you will be in one of two "page building modes": "Preview" or "Edit". By default, you will be in the "Edit" mode, but you can switch between the two at any time, simply by clicking on the respective button. Clicking on them right now does nothing, since you are not in the "My Site" section of the application.

Edit Mode

Preview Mode

In this mode, you can preview the webpage you are on as if it were a real physical page. Everything you do within the Edit mode will automatically show up on the preview mode.

Advanced Settings

At the end of the box on the bottom left, you will see an "Advanced..." button. Here, you will be able to have much more control over the look of your website, as well as access some extra settings.
