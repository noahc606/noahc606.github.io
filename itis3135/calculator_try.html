Note 1

I was able to build this calculator without the FCC tutorial.

Note 2

The Windows 10/11 Microsoft calculator gives results on the fly as you type in operators (+, -, etc). Thus, that calculator does not do order of operations unless you explicitly type in parentheses.

This calculator calculates whatever you put in it as a single expression, taking into account order of operations. As a result, it is comparable to the results you would get if you typed in a math expression into the Google search bar.
